# Description

Ce mod est le portage de la traduction de cultist simulator créée par Burton. La traduction couvre le jeu de base cultist simulator ainsi que le DLC "The Dancer"

# Installation

## Steam

Ce mod sera prochainement mis a disposition sur le steamworkshop, en attendant se référer a la procédure d'installation de GOG et autres.

## GOG et autres 

- Téléchargez le fichier zip de la dernière version du mod de traduction [ici](https://github.com/moare3788/cultist_simulator_french_traduction/archive/master.zip)
- Dézippez le zip téléchager
- Copiez le répertoire "Traduction" dans le répertoire "C:\{votre_utilisateur}\AppData\LocalLow\Weather Factory\Cultist Simulator\mods" (Attention le répertoire AppData est un répertoire masqué, il faudra donc afficher les répertoires cachés dans votre explorateur de fichier
- Lancez le jeu cultist simulator
- Dans le menu "The invisible separapeum", le mod "Traduction FR" devrait apparaitre, cliquez sur "enable"
- Redémarrez le jeu
- La langue francaise est désormais accessible dans le menu language
- Bon jeu ;)
